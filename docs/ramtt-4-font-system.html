Copy-paste tokens til implementering. Tailwind 4-kompatibel via @theme directive. 
/* ─── Font Families ─── */
--font-display: 'Cormorant Garamond', Georgia, serif;
--font-body:    'Satoshi', -apple-system, sans-serif;
--font-label:   'Space Grotesk', sans-serif;
--font-mono:    'JetBrains Mono', 'SF Mono', monospace;
--font-editorial: 'Lora', Georgia, serif;

/* ─── Space Grotesk Label Utilities ─── */
--label-section:  500 14px/1.2 var(--font-label); letter-spacing: 0.08em; text-transform: uppercase;
--label-default:  600 11px/1.2 var(--font-label); letter-spacing: 0.10em; text-transform: uppercase;
--label-micro:    600 10px/1.2 var(--font-label); letter-spacing: 0.12em; text-transform: uppercase;

/* ─── Cormorant Display Scale ─── */

06 — Performance & Loading
Fire fonte i woff2 er under 200 KB samlet, subsetted. Denne strategi sikrer at intet blokerer rendering.
Preload
Kun Satoshi preloades — det er den første font brugeren ser. Alle andre loader asynkront med font-display: swap.
Subset
Cormorant Garamond subsettes til Latin (evt. kun de tegn der bruges). Space Grotesk subsettes til uppercase Latin + tal. JetBrains Mono til tal + få tegn (:./−+).
Format
Alt i woff2. Ingen woff1/ttf fallbacks nødvendige for moderne browsers. Variable fonts hvor tilgængelige (Satoshi, Space Grotesk).
Self-Host
Self-host alle fonte — én DNS-request mindre pr. font vs. Google Fonts. Servér fra samme domæne/CDN som appen.
07 — Decision Guide
Når du er i tvivl om hvilken font du skal bruge, følg dette flowchart.
Er det et tal eller en dataværdi?
→ JetBrains Mono

Er det en label, tag, badge, section header, tab, eller kategori?
→ Space Grotesk · UPPERCASE · tracked

Er det en "special moment" — velkomst, milestone, onboarding, science-side?
→ Cormorant Garamond · wt 300 · oldstyle nums

Er det editorial/Letters brødtekst?
→ Lora

Alt andet?
→ Satoshi
